# Portafolio V4

Portafolio personal desarrollado con HTML, CSS, JavaScript y Tailwind CSS.
El sitio esta organizado por componentes y muestra secciones como inicio, habilidades, educacion, proyectos y contacto.

## Que incluye

- Carga dinamica de secciones HTML desde la carpeta `components/`
- Soporte de tema claro/oscuro con preferencia guardada en `localStorage`
- Formulario de contacto conectado con EmailJS
- Interacciones de UI para menu movil y filtros de habilidades

## Estructura general

- `index.html`: plantilla principal
- `script.js`: logica de carga de componentes e interacciones
- `styles.css`: estilos adicionales
- `components/`: secciones reutilizables del sitio
- `assets/`: recursos visuales

## Ejecucion local

Abre `index.html` en el navegador o usa una extension de servidor local (por ejemplo Live Server) para evitar problemas de rutas al cargar componentes por `fetch`.
